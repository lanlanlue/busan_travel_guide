## 🚨 時間管理風險 (HIGH PRIORITY)

### ⚠️ Day 1 週五 — 第一晚時間緊張

**現況分析：**

```
18:00 落地機場
→ 出關 (15-20分)
→ B1購卡+加值 (15-20分)
→ BGL搭乘 (20分) + 地鐵 (15分) = 35分
→ 步行至飯店 (5分)
總耗時：約70-90分鐘
→ 預計19:20-19:50抵達飯店
→ 洗漱稍整 (15-20分)
→ 20:45出發The Bay 101
```

**問題：**

- ✅ 時間**勉強可行**，但沒有任何容錯空間
- ⚠️ 若出關排隊時間過長或購卡遇冷場，會直接延後20-45分鐘
- ⚠️ 第一天時差調整+長途飛行，體力可能不足

**建議改進：**

1. **備案A：取消或推遲The Bay 101**
   - 改為：抵達飯店後簡單用餐，早點休息調時差
   - 將The Bay 101移到Day 2傍晚（若不排其他景點）
2. **備案B：簡化第一晚活動**
   - The Bay 101改為看看外景廣場+坐飯店客房鬆懈，不必特地出門

3. **改善Day 1準備：**
   - 建議：**提前在台灣研究Kakao T叫車**，不熟悉韓文介面會延誤
   - 預備：在台灣下載Naver Map離線地圖（飯店地址、景點地址中文標記）

### ⚠️ Day 3 週日 — 龍宮寺時間距離不確認

**現況安排：**

```
09:30-11:15 龍宮寺（機張）
11:20-12:20 龍宮寺附近午餐（2-3分步行）
12:35 抵達Skyline Luge售票處
13:00 Pass開卡（第一次嗶卡）
```

**疑慮點：**

- 龍宮寺步行下來 + 108階梯 + 拍照 = 通常需要1.5-2小時以上（非1小時45分）
- 午餐12:20出發 → 12:35抵達Luge ≈ 15分鐘車程
- **但龍宮寺步行出來（從海上觀音殿返回停車場）需時 10-15分鐘**
- 實際時間恐怕：12:20出發 → 12:50-13:00才抵達Luge

**風險：**

- ⚠️ 若龍宮寺遊玩時間超出（通常會），會直接影響到Luge等候時間
- 💡 **Pass開卡的13:00是硬時間點**，若遲到會推後整個Pass的計時到期（變成Day 5 14:00而非13:00）

**改善方案：**

1. **實測方案：**
   - 龍宮寺改為08:00出發（而非09:30）→ 08:30-10:45遊玩
   - 10:45出發至午餐地點 → 11:15開始用餐
   - 12:00出發 → 12:25抵達Luge，給15分鐘整理時間
   - 12:40左右嗶卡（給容錯空間）

2. **保險方案：**
   - 龍宮寺改為09:00出發（提早30分）
   - 餐廳改為Luge附近用餐（不在龍宮寺外頭浪費時間）


1. **甘川洞實際遊玩時間不足**

     - 韓服換裝 (25分)
     - 彩色階梯拍照 (20分)
     - 天空展望台 (25分)
     - **手翻書 (40-50分) ← 風險點**
     - 壁畫巷弄 (20分)
     - 還韓服 (5分)
   - 🚨 **手翻書工作室營業狀況不確定**（文案已標記需確認）
   - 🚨 **週日早上入村後人多，實際可能需要多預留15-30分鐘排隊**

2. **松島移動時間緊張**
   - 11:20出發 → 11:45抵達 = 25分鐘
   - 若甘川洞超時5分（變11:25出發），就變成11:50才抵達
   - 抵達後還要排隊找Pass兌換窗口 ← **最後10分鐘內完成兌換**
   - ⚠️ **極度危險！若排隊人多，難以在13:00前完成**

3. **無備方案**
   - 文案未提及若無法準時完成該怎麼辦
   - 若12:10內無法嗶卡，整個松島+龍宮雲橋會無法兌換 ❌

**Critical改善方案（必須實施之一）：**

**方案 A - 最保險（推薦）：**

```
Pass開卡改為 Day 4 13:30（海濱列車搭乘）
而非 Day 3 13:00（Luge搭乘）

這樣：
- Day 3-4 就已開始計時
- Day 5 到期時間 = Day 5 13:30
- 給甘川洞→松島多出30分鐘容錯空間
- 遠比目前安排安全
```

**方案 B - 重新安排Day 5：**

```

06:30 飯店出發
07:10 抵達甘川洞

10:30 出發松島
11:00 抵達松島
```

**方案 C - 取消手翻書或移至Day 2：**
